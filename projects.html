Projects

3D Photo Studio

This work aims at recovering high-fidelity surface models for human faces and small objects. Images are captured in a snapshot with well-designed modeling systems. I particitated in this project when I was in NJU doing my FYP with Prof. Xun Cao. Details could be found in the 3D studio project page.





Large-scale Multi-view Stereo

This work aims at developing an accurate, robust and efficient pipeline for large-scale multi-view stereo. The corresponding pipeline is benchmarked in the recent Tanks and Temple dataset and achieves the state-of-the-art performance. The algorithms have been integrated into the world's largest online 3D reconstruction platform Altizure.com to process more than 10,000 images per day.



MVSNet

This work aims at learning to estimate the dense representation given the multi-view images. We encode the camera geometry as the differentiable homography in the network to bridge the 2D image feature and 3D cost volume, which makes the end-to-end training practical. MVSNet is able to generate point cloud with comparable accuracy to the traditional state-of-the-arts while being significantly more complete. You may check out our code for MVSNet here.
